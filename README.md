# Nodejs rest api boilerplate
## Nodejs Express  


### Features

* Express
* `nodemon`
* `eslint-config-google`
* `docker`
* `docker-compose`


### Getting Started

* clone 

```sh
npm install
npm start
```

* docker

Build and run inside a docker container

```sh
npm run dbuild
npm run drun 
```

* Test 
```sh
npm test
```
